### 1. **JSX (JavaScript XML)**  
   - A syntax extension that lets you write HTML-like code inside JavaScript.  
   - Example: `<h1>Hello World</h1>` is JSX.

---

### 2. **Components**  
   - Building blocks of a React application. They can be:
     - **Functional Components**: Use functions to define UI.
     - **Class Components**: Use ES6 classes (less common now).  
   - Example:
     ```jsx
     function MyComponent() {
       return <h1>Hello!</h1>;
     }
     ```

---

### 3. **Props (Properties)**  
   - Used to pass data from parent to child components.  
   - Example:
     ```jsx
     <MyComponent name="John" />
     ```

---

### 4. **State**  
   - A component's local memory for managing dynamic data.  
   - Use `useState` hook in functional components.  
   - Example:
     ```jsx
     const [count, setCount] = useState(0);
     ```

---

### 5. **Hooks**  
   - Functions to use state and lifecycle in functional components.  
   - Common hooks:
     - `useState`: Manage state.
     - `useEffect`: Handle side effects (e.g., fetching data).
     - `useContext`: Share state globally.

---

### 6. **Lifecycle Methods**  
   - Used in class components for different phases of a component's life.  
   - Replaced by hooks like `useEffect` in functional components.

---

### 7. **Context API**  
   - For managing global state without prop drilling (passing props deeply).  
   - Example:
     ```jsx
     const MyContext = React.createContext();
     ```

---

### 8. **React Router**  
   - For handling navigation and routing in a React app.  
   - Example: `/home`, `/about`.

---

### 9. **Redux / State Management Libraries**  
   - Manage complex global states.  
   - Redux, Zustand, or React Query are popular choices.

---

### 10. **Rendering Types**  
   - **Client-Side Rendering (CSR)**: Updates the UI in the browser.  
   - **Server-Side Rendering (SSR)**: Renders pages on the server for faster load times.  
   - **Static Site Generation (SSG)**: Pre-builds pages during deployment.

---

### 11. **React DevTools**  
   - A browser extension to inspect React components and debug.  

These are foundational for building and scaling React apps efficiently.
